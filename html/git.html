Git

  • git add- добавление изменений в индекс
  • git status -показывает текущее состояние изменений
  • git commit -m "сообщение" - фиксирует изменения в репозитории с сообщением
  • git push - отправляет локальные коммиты в удаленные репозитории
  • git pull- забирает изменения из удаленного репозитория
